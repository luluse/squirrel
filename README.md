# React project

## Overview
 A collection of my friends. Search bar helps me narrowing down my search when looking for a friend in the list.  

## User Stories
- I created an API where I stored all my friends information and I want an interface to access my data
- I want to be able to access the full list of my friends in one place
- I want to be able to quick access one particular fiend by typing their name in search bar

## Tools
- React
- Redux
- Hooks
- Tachyons
- deployed on gh-pages

### [deployed app](https://luluse.github.io/squirrel/)

## Data Flow
- User types in search box the name of friend they're looking for
- For each input list updates according to the input
- For each input, request is sent the API and API sends back data matching the input
- List updates automatically after each input for quick access of data for user
